# Ollama Auth Proxy

Прокси-сервер на **FastAPI**, который добавляет авторизацию по токену для локального сервера [Ollama](https://ollama.ai/). 

По умолчанию Ollama не имеет встроенной аутентификации. Этот микросервис решает данную проблему: он принимает запросы, проверяет `Bearer` токен и, если токен верный, прозрачно перенаправляет запрос в Ollama, возвращая ответ в потоковом режиме (Streaming) без таймаутов.

## Особенности
- **Безопасность**: Защита API с помощью статического `Bearer` токена.
- **Потоковая переда��а (Streaming)**: Поддерживает потоковую генерацию текста.
- **Отсутствие таймаутов**: Специально настроен для работы с тяжелыми моделями, которые долго генерируют первый токен.
- **Прозрачность**: Тело запроса, заголовки и параметры передаются в Ollama без искажений.

## Требования
- Python 3.8+
- Запущенный сервер Ollama (по умолчанию на `http://localhost:11434`)

## Установка и запуск

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/KKittyCatik/server_auth_ollama.git
   cd server_auth_ollama
   ```

2. **Создайте и активируйте виртуальное окружение:**
   ```bash
   python -m venv venv
   source venv/bin/activate  # Для Linux/macOS
   # venv\Scripts\activate   # Для Windows
   ```

3. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Настройте переменные окружения:**
   Скопируйте файл с примером переменных окружения:
   ```bash
   cp .env.example .env
   ```
   Откройте файл `.env` и задайте свой секретный токен:
   ```env
   SECRET_TOKEN=ваш_супер_секретный_токен
   OLLAMA_BASE_URL=http://localhost:11434
   ```

5. **Запустите сервер:**
   ```bash
   uvicorn main:app --host 0.0.0.0 --port 8000
   ```

## Использование

Теперь ваше защищенное API доступно на `http://localhost:8000`. При каждом запросе необходимо передавать заголовок `Authorization: Bearer <ваш-токен>`.

### Пример запроса (cURL)
```bash
curl -X POST http://localhost:8000/api/generate \
  -H "Authorization: Bearer ваш_супер_секретный_токен" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "llama2",
    "prompt": "Напиши короткий стих про космос"
  }'
```
